# Technical Test by [muatmuat](https://muatmuat.com/).

## Installation

```bash
git clone https://github.com/mariio46/technical-test-muatmuat.git
```

```bash
cd technical-test-muatmuat
```

```bash
npm install
```

```bash
npm run dev
```
